== Description ==
SEO Keyword Interlinker is a WordPress plugin that generates keyword link for SEO enhancement. It also creates a title tag to the link and has an option to link in new _blank tab or _self.

=== IMPORTANT ===
SEO Keyword Interlinker is a free plugin that allows user to automatically generate a internal link for the specified keyword or phrase, with option for new tab or self tab and position.

=== HOW IT WORKS ===
The plugin allows the user to specify a list of focus phrases of keywords to be automatically generate an internal link.

=== FOR DEVELOPER ===
The plugin can be tweaked and reference updates enhanced.
